# Air Quality UI

## Local Development Setup

From within air-quality-ui:

### Install
`npm install`

### Create .env file with content
```
VITE_AIR_QUALITY_API_URL=http://localhost:8000
VITE_AIR_QUALITY_UI_URL=http://localhost:5173
```

### Start
`npm run dev`

### Unit Test
`npm run test`

### Lint (with fix)
`npm run lint:fix`
